# 🎬 YouTube Clone – HTML & CSS Crash Course Project

A simple **YouTube homepage clone** built using only **HTML and CSS** as part of a crash course project. This project was created for practice purposes to improve layout building, positioning, and responsive design skills.

---

## 📌 Project Overview

This project replicates the basic layout of YouTube’s homepage, including:

- Navigation bar (header)
- Sidebar menu
- Video grid layout
- Video thumbnails
- Channel info section
- Basic responsive layout structure

It focuses purely on **frontend design (no JavaScript or backend)**.

---

## 🎯 Purpose of the Project

- Practice HTML structure and semantic tags
- Improve CSS layout skills (Flexbox / Grid)
- Learn positioning and spacing techniques
- Understand real-world UI cloning
- Build frontend discipline through replication

---

## 🛠️ Technologies Used

- HTML5
- CSS3
